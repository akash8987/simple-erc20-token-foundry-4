# Simple ERC-20 Token

A high-quality, beginner-friendly starting point for deploying a standard ERC-20 token on Ethereum or any EVM-compatible blockchain. This repository utilizes the industry-standard OpenZeppelin libraries to ensure security and reliability.

## Features
- **Standard Compliance**: Fully compatible with the ERC-20 interface.
- **Security**: Built using OpenZeppelin's battle-tested `ERC20.sol`.
- **Developer Friendly**: Flat structure for easy navigation and rapid prototyping.

## Quick Start
1. Install Foundry.
2. Run `forge build` to compile the contracts.
3. Run `forge test` to execute the included test suite.
